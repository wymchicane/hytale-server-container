# Installation

Deploy your Hytale server in minutes. Follow these steps to get up and running with a lightweight, production-ready Docker container.

## Prerequisites

Before installing, make sure you have:

- **Hardware**: x86_64 or ARM64 machine (Raspberry Pi 4+ recommended for ARM)
- **OS**: Linux, Windows WSL2, macOS, or any OS that supports Docker
- **Docker**: Latest version installed ([Install Guide](https://docs.docker.com/get-docker/))
- **Hytale License**: A valid Hytale account is required for server authentication
